Código Incompleto



Este erro ocorre quando fazemos o algoritmo pela metade, ou seja, faltando algum pedaço.


Exemplos:

ERRADO

 
//Descricao do programa: Escreva um algoritmo que receba 5 números e mostre a soma dos mesmos. 

programa
{
	funcao inicio()
	{
		inteiro num, soma=0, i
		
		para (i=1; i<5; i++) {
		escreva ("Informe o número que deseja acrescentar a soma : ")
		leia (num)
		soma = num + soma
		}
	}  //Perceba que o usuário escreveu um script que lê os 5 números e os acumula em uma variável (soma),
	   //contudo ele não mostrou a soma dos números como  questão pedia. 
}

CERTO

 
//Descricao do programa: Escreva um algoritmo que receba 5 números e mostre a soma dos mesmos. 

programa
{
	funcao inicio()
	{
		inteiro num, soma=0, i
		
		para (i=1; i<5; i++) {
		escreva ("Informe o número que deseja acrescentar a soma : ")
		leia (num)
		soma = num + soma
		}

		escreva ("A soma dos números recebidos é : ", soma) //Agora está correto. 
	}
}
