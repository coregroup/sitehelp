Inicio Contatos Duvidas Sobre

Usar sinal de atribuicao (=) em vez de comparacao (==), ou vice-versa:



O comando de atribuicao de valor de expressoes a variaveis eh o sinal de igual (=). O comando de comparacao (o teste "se o conteudo da variavel eh igual a expressao") eh feito com dois sinais de igual (==).


Exemplos:

ERRADO

 

//Descricao do programa: Leia o numero 5 e informe se o numero digitado pelo usuario foi igual a 5. 
programa
{
	funcao inicio()
	{
		inteiro numero		
		escreva("Digite o numero 5: ")
		leia(numero)
		se (numero=5){    /* A variavel "numero" recebe o valor 5 em vez de compara-la com o numero 5.
Mesmo o usuario digitando outro numero a variavel "numero" sempre recebera o valor 5.*/ 
			escreva("Voce digitou o numero 5") // o programa sempre vira para essa linha, independetemente do numero digitado.
		} 
		senao escreva("voce nao digitou o numero 5")  // essa linha nunca sera executada. 		
	}
}

CERTO

 
//Descricao do programa: Leia o numero 5 e informe se o numero digitado pelo usuario foi igual a 5. 
programa
{
	funcao inicio()
	{
		inteiro numero		
		escreva("Digite o numero 5: ")
		leia(numero)
		se (numero==5){    //a variavel "numero" eh comparada com o numero 5. 
			escreva("Voce digitou o numero 5") ////se voce digitar o numero 5 o programa passa para essa linha.   
		} 
		senao escreva("voce nao digitou o numero 5")  // se digitar um numero diferente de 5 o programa pula para essa linha.  		
	}
}

Erros:

    1. Usar sinal de atribuicao (=) em vez de comparacao (==), ou vice-versa.
    2. Utilizar parentesis Descasados .
    3. Utilizar chaves Descasados .
    4. Utilizar variaveis nao declaradas .
    5. Colocar espaco entre os dois caracteres que compoem os operadores ( ==, <=,>=,!=) .
    6. Inversao da ordem dos dois caracteres que compoem os operadores (<=,>=,!=) escrevendo (=<,=>,=!)
    7. Dividir um numero inteiro por um numero real .
    8. Usar as aspas Descasadas num Printf .
    9. Utilizar as aspas Descasadas num Scanf .
    10. Errar a escrita de um comando .
    11. esquecer o E comercial (&) no Scanf .
    12. Colocacao das flags incorretas (%c,%f,%d,...) no Scanf ou Printf .
    13. Criacao de ciclo infinito.
    14. Utilizacao de barra (/) em vez de barra invertida (\).
    15. Declaracao de variavel incompativel com o seu tipo declarado.
    16. Fazer comparacao com uma variavel do tipo char e um caracter e nao colocar este caracter entre aspas simples.
    17. Uso incorreto das chaves em comandos de repeticao.
    18. Utilizacao incorreta de variaveis contadoras ou acumuladoras em comandos de repeticao (Esquecer de declarar, ...).
    19. Erro na declaracao de variaveis (Variaveis com espacos, comecar com um numero, uso de acentuacao grafica ao declarar variaveis, etc.).
    20. Erros de Case Sensitive.
    21. Uso de codigo que nao eh necessario para a solucao do problema
    22. Codigo Incompleto.
    23. Erro Semantico
    24. Codigo nao corresponde com a questao.
    25. Utilizacao de variavel sem necessidade.
