Fazer comparação com uma variável do tipo char ou cadeia e não colocá-lo em aspas .



Um erro que acontece quando esquecemos de colocar as aspas em uma comparação entre caracteres ou variáveis.


Exemplos:

ERRADO

 

//Descricao do programa: Escreva um algoritmo que receba uma senha e informe se a senha é igual a "casa". 
//Se sim, ele exiba "Bem vindo"; senão : "Senha Incorreta". 

programa
{
	funcao inicio()
	{
		cadeia senha

		escreva ("Informe a senha por favor : ")
		leia (senha)

		se (senha == casa) escreva (" Bem Vindo")
		senao escreva ("Senha Incorreta")
	} // Perceba que na tag de repetição "se", a cadeia de caracteres "casa" não estava entre aspas, 
	  //o que causou um erro no script.
}
