Utilizar chaves descasadas:



Esse tipo de erro é chamado de erro de sintaxe, são erros de digitação ou outro erro no código que bloqueiam a execução do programa. Todos as chaves abertas, têm que ser fechadas para que o compilador possa compilar o codigo. O proprio compilador informa onde ocorreu o erro.


Exemplos:

ERRADO

 
//Descricao do programa: Leia a idade do usuario e informe se ele pode comprar bebidas alcoolicas. 
programa
{
	funcao inicio()
	{
		inteiro idade		
		escreva("Qual a idade: ")
		leia(idade)
		se (idade>=18){    // As chaves devem ser fechadas para que o programa possa ser compilado. 
			escreva("Usuario maior de idade, pode comprar!!") 
			// O erro ocorre nessa linha por falta de uma chave(}). 
		senao { 
			escreva("Usuario menor de idade, nao pode comprar.")  		
		}
	}
}
