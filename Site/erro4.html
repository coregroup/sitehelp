Inicio Contatos Duvidas Sobre

Nao declarar variaveis utilizadas dentro do programa.



Todas as variaveis devem ser declaradas. Esse tipo de erro eh chamado de Erro de Compilacao.
No exemplo abaixo, a variavel "area", foi utilizada mas nao foi declarada.


Exemplos:

ERRADO

 
//Descricao do programa: Calcular a area de um retangulo. 
programa
{
	funcao inicio()
	{		
		real altura, largura   // A variavel "area" nao foi declarada.	
		escreva("Digite a altura: ")  
		leia(altura)
		escreva("Digite a largura: ") 
		leia(largura)
		area=altura*largura  //A variavel "area" nao foi declarada.
		escreva("Area total: ", area)
	}
}

CERTO

 
//Descricao do programa: Calcular a area de um retangulo. 
programa
{
	funcao inicio()
	{		
		real altura, largura, area 	
		escreva("Digite a altura: ")  
		leia(altura)
		escreva("Digite a largura: ") 
		leia(largura)
		area=altura*largura 
		escreva("Area total: ", area)
	}
}
