Erro na declaração de variáveis, relacionados ao nome !



Este erro acontece devido ao motivo que o Portugol Studio segue regras de que :

  • O primeiro caracter deve ser uma letra;
  • Os nomes devem ser formados por caracteres pertencentes ao seguinte conjunto: {a,b,c,...,z,A,B,C,...Z,0,1,2,...,9,_};
  • Não deve haver espaço em branco;
  • Não deve haver variáveis repetidas;
  • Cuidado com Case Sensitive.

Exemplos:

ERRADO

 
//Descricao do programa: Escreva um algortmo que receba o nome de um usuário e dê bom dia ao nome do usuário. 

programa
{
	funcao inicio()
	{
		cadeia nome do usuário 
		
		//Observe que o usuário criou uma variável com espaços e contendo um acento no nome "usuário", claro que o resultado foi um erro.
		//Isso porque o Portugol Studio reconheceu "nome", "de", "usuário", como 3 variáveis diferentes. Uma solução seria unir as 3 palavras utilizando urderline's(_), 
		// menos (-), etc; Como também se poderia resumir onome em algumas letras : ndu.

		escreva ("Informe o nome do usuário :")
		leia (nome do usuário)
		escreva ("Bom dia ", nome do usuário, " !")
	}
}
