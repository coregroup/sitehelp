Erros relacionados ao Case Sensitive



Este erro acontece devido ao motivo que o Portugol Studio segue regras de que :

  • O primeiro caracter deve ser uma letra;
  • Os nomes devem ser formados por caracteres pertencentes ao seguinte conjunto: {a,b,c,...,z,A,B,C,...Z,0,1,2,...,9,_};
  • Não deve haver espaço em branco;
  • Não deve haver variáveis repetidas;
  • Cuidado com Case Sensitive.

Case Sensitive : termo em inglês que significa Sensível ao tamanho, no caso, o tamanho refere-se a maiúsculas e minúsculas na definição de variáveis, métodos e comando em um programa ou um compilador. Ou seja, significa também que o Português Estruturado faz a distinção entre os tamanhos das letras.

 

Exemplos:

ERRADO

 
//Descricao do programa: Escreva um script que dê um desconto de 10% no valor do produto. 

programa
{
	funcao inicio()
	{

		Real valor_do_produto, Desconto //Este erro é válido tanto para comandos como para nome de variáveis. 

		escreva ("Informe o valor do produto : ")
		leia (valor_do_produto)

		desconto = valor_do_produto-((valor_do_produto/100)*10)
		
		Escreva ("O valor do produto com o desconto é : ", desconto)
	}
}
